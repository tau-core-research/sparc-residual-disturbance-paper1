# Illustrative Rotation-Curve Points

This directory contains the minimal derived point table used only to render the contextual rotation-curve figure in the Paper 1 arXiv manuscript.

Files:

- `paper1_context_rotation_points.csv`

The table contains NGC2403 as an external class-A regular-disk example and NGC1705 as an external class-C disturbed-dwarf example. The rows combine SPARC rotation-curve columns with already-derived per-radius residual diagnostics from the Paper 1 packet.

The figure generated from these rows is visual context only. It shows how the Paper 1 external-label contrast appears in rotation-curve space and must not be interpreted as an additional tuned endpoint or independent validation.
